# KrisLiDev LinkTree

Uma página estilo Linktree criada para centralizar meus links profissionais, projetos e formas de contato em um único lugar.

A proposta do projeto é ser uma página simples, rápida e visualmente agradável, funcionando como um pequeno hub pessoal para apresentar meu trabalho e facilitar o acesso às minhas redes e projetos.

## Preview

A página inclui:

* Links para redes profissionais
* Seção de projetos
* Pequena apresentação pessoal
* Layout simples e responsivo

## Tecnologias utilizadas

* HTML5
* CSS3

## Estrutura do projeto

```
KrisLiDev-LinkTree
│
├── index.html
│
├── css
│   ├── reset.css
│   └── style.css
│
└── assets
    └── imagens / gifs
```

## Objetivo do projeto

Este projeto também funciona como um espaço para experimentar ideias de design e melhorar minhas habilidades em desenvolvimento front-end, especialmente na criação de interfaces simples e bem organizadas.

Com o tempo, novas melhorias visuais e estruturais podem ser adicionadas conforme eu continuo estudando e evoluindo como desenvolvedor.
